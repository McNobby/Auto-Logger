COMMANDS AND FEATURES

FEATURES

The bot’s primary function is to detect, remove, mute and log users that include slurs in their messages, in the fastest most clean way.


Auto Logger is designed to ignore spaces so it can detect slurs written like this:

“S L U R”.

Whenever someone is muted they will get a fancy direct message from the bot saying they were muted. The same happens whenever they are unmuted

Auto Logger prioritises a un-cluttered chat, so all error messages will be sent in dm's of the one executing the command.

COMMANDS

!mute @member

!unmute @member

SETTING UP

For Auto Logger to work you'll need a few things to get started.

Action Log

You'll need a action log channel where Auto Logger will log all actions taken. Your staff can also use this channel to log other actions aswell.

Deletion log

Then you'll need a Deletetion log channel where Auto Logger will log all deleted messages.

Staff Role

Lastly you will need a staff role. People with this role will have acsess to all of Auto Loggers commands, exept admin commands.


COMMANDS


!setup alog #channel - Will define what channel to use for action log

!setup dlog #channel - Will define what chennel to use for deletion log

!setup sRole @role - will define staffrole
